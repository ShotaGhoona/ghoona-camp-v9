# テーマカラー設定 - 実装レポート

## 概要
Slack風のテーマカラー切り替え機能を実装

## 機能
- 5色のプリセット（パープル、レッド、イエロー、ブルー、グリーン）
- ライト/ダーク/システムモード対応
- ライト・ダークで異なるprimary色（ダークは暗め）
- localStorage永続化
- FOUC防止

## 新規作成ファイル
```
features/core/theme/
├── constants/
│   └── theme-presets.ts      # プリセット定義
├── lib/
│   ├── theme-applier.ts      # CSS変数適用
│   ├── theme-context.tsx     # React Context & Provider
│   ├── theme-init-script.ts  # FOUC防止スクリプト
│   └── theme-storage.ts      # localStorage操作
├── model/
│   └── types.ts              # 型定義
└── ui/
    └── ThemeSheet.tsx        # 設定Sheet UI

app/(authenticated)/test/theme/page.tsx  # テストページ
```

## 変更ファイル
- `app/layout.tsx` - FOUC防止スクリプト埋め込み
- `app/providers.tsx` - ThemeProvider追加
- `widgets/.../AppHeader.tsx` - テーマ設定ボタン追加（ドロップダウン内）

## 使用方法
1. ヘッダーのユーザーメニュー → 「テーマ設定」
2. 右からSheetがスライドイン
3. カラーとモードを選択

## テスト
`/test/theme` でコンポーネントの色確認可能
