# Ghoona Camp - Value Objects

## Overview
実装に迷わないようにビジネス要件に基づいたValue Objectsを定義します。
オーバーエンジニアリングを避け、実際のビジネス制約がある項目のみを対象とします。

## User Domain

### UserStatus
**概要**: ユーザーのアカウント状態
**値**: active, inactive
**制約**: アクティブなユーザーのみがサービスを利用可能

### Platform
**概要**: ソーシャルリンクのプラットフォーム種別
**値**: twitter, instagram, github, linkedin, facebook, youtube, website, blog, portfolio, other
**制約**: 定義されたプラットフォームのみ登録可能

### PublicFlag
**概要**: 公開設定のフラグ
**値**: true（公開）, false（非公開）
**制約**: デフォルトは公開設定

## Event Domain

### EventType
**概要**: イベントタイプの分類
**値**: general, study, exercise, meditation, discussion, presentation, workshop
**制約**: 定義されたタイプのみ作成可能

### ParticipantStatus
**概要**: イベント参加者の状態
**値**: registered（参加登録済み）, cancelled（キャンセル）
**制約**: キャンセル後の再登録は可能

## Title Domain

### TitleLevel
**概要**: 称号のレベル（1-8の固定値）
**値**: 
- Level 1: まどろみ見習い（Sleeper） - 1日
- Level 2: 早起き候補生（Early Bird Candidate） - 7日
- Level 3: 朝活探検家（Morning Explorer） - 30日
- Level 4: 朝の住人（Morning Resident） - 100日
- Level 5: 夜明けの戦士（Dawn Warrior） - 200日
- Level 6: 朝光の使者（Morning Light Messenger） - 365日
- Level 7: 暁の守護者（Dawn Guardian） - 500日
- Level 8: 朝活の伝説（Morning Legend） - 1000日
**制約**: 1-8の範囲でのみ有効、参加日数に応じて獲得

### ActiveFlag
**概要**: エンティティの有効/無効状態
**値**: true（有効）, false（無効）
**制約**: 無効なエンティティは表示・操作対象外

### CurrentFlag
**概要**: 現在設定中の称号かどうか
**値**: true（現在設定中）, false（設定していない）
**制約**: ユーザーは1つの称号のみを現在設定可能

## Notification Domain

### NotificationType
**概要**: 通知の種別
**値**: achievement（称号獲得）, reminder（リマインダー）, rival_update（ライバル更新）, event（イベント関連）
**制約**: ユーザーの通知設定に応じて送信制御

## Business Rules

### 制約事項
- ライバル登録は1人につき最大3人まで
- 称号レベルは1-8の固定値
- イベント開催時間は start_time < end_time
- 目標の有効期間は ended_at > started_at
- 自分自身をライバルに設定不可
- ユーザーは1つの称号のみを現在設定可能

### 将来検討項目
以下は現在のスコープ外だが、将来的にValue Objectが必要になる可能性がある項目：
- 参加時間の最小値チェック（Discord連携時）
- タイムゾーンバリデーション
- スキル・興味のカテゴリ制限
