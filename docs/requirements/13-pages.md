# Ghoona Camp - Pages Design

## Overview
Ghoona Campアプリケーションで必要なページ一覧と各ページの機能要件です。

## Authentication Pages
Clerk認証を使用したログイン関係のページ

### 1. Landing Page (LP)
**パス**: `/`  
**アクセス**: 未認証ユーザー

**機能:**
- サービス概要・価値提案
- 朝活コミュニティの魅力紹介
- 称号システムの説明
- ユーザー体験談・実績
- サインアップへの導線

### 2. Sign In Page
**パス**: `/sign-in`  
**アクセス**: 未認証ユーザー

**機能:**
- Clerk認証画面
- ログイン後はダッシュボードへリダイレクト

### 3. Sign Up Page
**パス**: `/sign-up`  
**アクセス**: 未認証ユーザー

**機能:**
- Clerk認証画面
- 新規登録後はプロフィール設定へリダイレクト

## Main Application Pages
認証済みユーザー向けのメインアプリケーション

### 4. Dashboard Page
**パス**: `/dashboard`  
**アクセス**: 認証済みユーザー

**機能:**
- 今日の朝活状況（参加済み/未参加）
- 自分の統計サマリー（連続日数、総参加日数、現在の称号）
- ライバルの最新状況
- 本日のイベント一覧
- 最近の通知
- 目標の進捗状況

### 5. Ranking Page
**パス**: `/ranking`  
**アクセス**: 認証済みユーザー

**機能:**
- 今月のランキング（参加日数順）
- 総合ランキング（総参加日数順）
- 連続日数ランキング
- ランキング期間切り替え
- 自分の順位ハイライト
- ライバルの順位表示

### 6. Activity Log Page
**パス**: `/activity`  
**アクセス**: 認証済みユーザー

**機能:**
- カレンダー形式の参加履歴表示
- 参加した日の詳細（参加時間、イベント情報）
- 月次・週次の統計グラフ
- 目標達成状況の可視化
- 参加ログの詳細フィルタリング

### 7. Events Page
**パス**: `/events`  
**アクセス**: 認証済みユーザー

**機能:**
- 開催予定イベント一覧
- イベント詳細表示
- イベント参加申込・キャンセル
- 自分が作成したイベント管理
- 新規イベント作成
- 参加者一覧表示

### 8. Goals Page
**パス**: `/goals`  
**アクセス**: 認証済みユーザー

**機能:**
- 自分の目標一覧表示
- 目標の作成・編集・削除
- 目標の公開/非公開設定
- 目標の進捗管理
- 他のユーザーの公開目標閲覧

### 9. Titles Page
**パス**: `/titles`  
**アクセス**: 認証済みユーザー

**機能:**
- 全称号一覧（8段階）
- 各称号の詳細情報・ストーリー
- 獲得条件・必要日数表示
- 自分の獲得済み称号
- 称号保持者一覧
- 次の称号までの進捗

### ✅10. Members Page
**パス**: `/members`  
**アクセス**: 認証済みユーザー

**機能:**
- メンバー一覧表示
- メンバー検索・フィルタ
- スキル・興味での絞り込み
- メンバープロフィールポップアップ表示
- ライバル設定・解除（ポップアップ内）

## Settings Pages
ユーザー設定関連のページ

### 11. Profile Settings Page
**パス**: `/settings/profile`  
**アクセス**: 認証済みユーザー

**機能:**
- 基本プロフィール編集
- プロフィール画像アップロード
- 表示名・一言プロフィール設定
- 自己紹介文章編集
- スキル・興味の登録
- SNS・外部リンク管理
- タイムゾーン設定

### 12. Vision Settings Page
**パス**: `/settings/vision`  
**アクセス**: 認証済みユーザー

**機能:**
- ビジョン・将来の目標設定
- ビジョン編集・更新

### 13. Notification Settings Page
**パス**: `/settings/notifications`  
**アクセス**: 認証済みユーザー

**機能:**
- 通知種別のON/OFF設定
- リマインダー時刻設定
- 称号獲得通知設定
- ライバル更新通知設定
- イベント通知設定

### 14. Account Settings Page
**パス**: `/settings/account`  
**アクセス**: 認証済みユーザー

**機能:**
- Clerkアカウント設定
- パスワード変更
- メールアドレス変更
- Discord連携設定
- アカウント削除

## Error Pages

### 15. 404 Not Found Page
**パス**: `/404`

### 16. 500 Error Page
**パス**: `/500`

### 未認証ユーザー

### 認証済みユーザー  
- 上記以外の全ページ
