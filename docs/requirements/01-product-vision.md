# Ghoona Camp - Product Vision

## Concept
朝活の仲間と一緒に成長する、コミュニティ重視のオンライン朝活サービス

## Basic Model
- **開催時間**: 毎朝6:00-7:00（1時間）
- **参加形式**: Discord上での共同作業時間
- **コミュニティ**: 継続的な仲間との繋がりとサポート
- **料金**: 完全無料
- **管理**: アプリで参加ログを自動記録・管理

## Target
20代〜30代のビジネスパーソン・学生・主婦で、朝の時間を有効活用したい人

## Differentiation
1. **仲間との繋がり**: 一人では続かない朝活を仲間と一緒に実現
2. **コミュニティサポート**: お互いの目標や進捗を共有し合える環境
3. **継続のための仕組み**: ソーシャルな繋がりによる自然な継続動機

## Core Features
1. 毎朝のDiscord朝活セッション
2. 参加者同士の交流・励まし合い
3. 自動参加ログ記録・管理
4. 目標設定と進捗共有
5. コミュニティ内での情報交換

## Value Proposition
- **参加者**: 朝型生活習慣 + 仲間との成長体験
- **コミュニティ**: 互いに支え合う持続可能な関係性
- **社会**: 健康的で生産性の高いライフスタイルの普及

**「一人では続かない朝活を、仲間と一緒に楽しく継続する」**コミュニティサービス
